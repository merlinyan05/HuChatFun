# HuChatFun 项目目录结构

## 最终完整结构

```
HuChatFun/
│
│  .gitignore
│  README.md                          # 项目说明
│  requirements.txt                   # Python 依赖
│
├─ corpus/                            # ① 原始语料（只读，不修改）
│  └─ HuChenFeng-1.1/
│     ├─ 2023年03月/
│     │    2023-03-10.md
│     │    ...
│     ├─ ...
│     └─ 2025年09月/
│          2025-09-14.md
│
├─ pipeline/                          # ② 数据处理脚本（按阶段编号）
│     step0_explore.py                # 探查语料格式
│     step1_rough_clean.py            # 粗切去噪
│     step2_segment.py                # 按对话切分
│     step3_score_filter.py           # 质量评分过滤
│     step4_build_pairs.py            # 构造训练对（含 LLM 调用）
│     step5_enhance.py                # 风格增强（可选）
│     step6_finalize.py               # 格式化输出 train/eval
│     run_all.sh                      # 一键跑完全流程
│     utils.py                        # 共用工具函数
│
├─ data/                              # ③ 数据产物（按阶段递进）
│  ├─ step1_cleaned/                  # 粗切后的文本
│  │    2023-03-10.txt
│  │    ...
│  │    _stats.json                   # 每阶段都带统计
│  │
│  ├─ step2_segments/                 # 切分后的结构化段落
│  │    conversations.jsonl
│  │    monologues.jsonl
│  │    _stats.json
│  │
│  ├─ step3_filtered/                 # 评分过滤后
│  │    high.jsonl
│  │    medium.jsonl
│  │    _stats.json
│  │
│  ├─ step4_pairs/                    # 训练对
│  │    conv_pairs.jsonl              # 连麦对话 → 问答对
│  │    mono_pairs.jsonl              # 独白 → LLM配问题
│  │    _stats.json
│  │
│  ├─ step5_enhanced/                 # 风格增强版（可选）
│  │    enhanced.jsonl
│  │
│  └─ final/                          # ⭐ 最终训练数据
│       train.json                    # 90%
│       eval.json                     # 10%
│       golden_quotes.json            # 手动精选金句（可选）
│       _summary.json                 # 数据集摘要统计
│
├─ training/                          # ④ 训练相关
│     train.py                        # 主训练脚本（QLoRA）
│     config.yaml                     # 训练超参数
│     merge_lora.py                   # 合并 LoRA 权重
│     convert_gguf.sh                 # 转 GGUF 格式
│
├─ eval/                              # ⑤ 评估
│     auto_eval.py                    # 自动评估（口头禅命中率等）
│     blind_test.py                   # 人工盲测生成
│     prompts_test.py                 # Prompt A/B 测试
│     test_questions.json             # 标准测试问题集
│     catchphrases.json               # 口头禅词库
│
├─ deploy/                            # ⑥ 部署
│     Modelfile                       # Ollama 模型配置
│     openclaw_config.yaml            # OpenClaw 接入配置
│     api_test.sh                     # API 测试脚本
│
├─ models/                            # ⑦ 模型文件（gitignore，不入库）
│  ├─ base/                           # 基座模型下载位置
│  │    qwen2.5-7b-instruct/
│  │
│  ├─ lora/                           # LoRA 训练输出
│  │  ├─ v1/
│  │  │    adapter_config.json
│  │  │    adapter_model.safetensors
│  │  │    training_args.json
│  │  └─ v2/
│  │
│  ├─ merged/                         # 合并后的全量模型
│  │    huchatfun-v1-merged/
│  │
│  └─ gguf/                           # 量化后的部署模型
│       huchatfun-v1-Q4_K_M.gguf
│       huchatfun-v1-Q5_K_M.gguf
│
├─ logs/                              # ⑧ 日志与实验记录
│     experiments.md                  # 每次训练的记录
│     wandb/                          # WandB 本地缓存
│
└─ docs/                              # ⑨ 文档
      format_notes.md                 # 阶段0 语料格式探查记录
      cleaning_strategy.md            # 清洗策略说明
      implementation_guide.md         # 完整实施指南
```

## 实际操作命令序列

```bash
# 1. 初始化项目
mkdir -p HuChatFun/{pipeline,data,training,eval,deploy,models,logs,docs}
cd HuChatFun
git init

# 2. 搬入语料（假设原来在同级目录）
mv ../HuChenFeng-1.1 corpus/

# 3. 探查语料
python pipeline/step0_explore.py
# → 产出 docs/format_notes.md

# 4. 跑数据管线
python pipeline/step1_rough_clean.py     # → data/step1_cleaned/
python pipeline/step2_segment.py         # → data/step2_segments/
python pipeline/step3_score_filter.py    # → data/step3_filtered/
python pipeline/step4_build_pairs.py     # → data/step4_pairs/
python pipeline/step6_finalize.py        # → data/final/

# 5. 检查数据
wc -l data/final/train.json             # 看行数
head -c 2000 data/final/train.json      # 看前几条

# 6. 下载基座模型（在训练机上）
huggingface-cli download Qwen/Qwen2.5-7B-Instruct \
    --local-dir models/base/qwen2.5-7b-instruct

# 7. 训练
CUDA_VISIBLE_DEVICES=0 python training/train.py

# 8. 合并 + 量化
python training/merge_lora.py
bash training/convert_gguf.sh

# 9. 部署测试（在 Mac Mini 上）
ollama create huchatfun -f deploy/Modelfile
ollama run huchatfun

# 10. 评估
python eval/auto_eval.py
python eval/prompts_test.py
```
